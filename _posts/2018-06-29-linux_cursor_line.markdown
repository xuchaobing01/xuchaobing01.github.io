>看一个真正的专家操作命令行绝对是一种很好的体验-光标在单词之间来回穿梭，命令行不同的滚动。 
 在这里强烈建立适应GUI节目的开发者尝试一下在提示符下面工作。 
 但是事情也不是那么简单，还是需要知道“如何去做”。在单词之间跳转，使用Ctrl+左右键。

- 常用
```
Ctrl+左右键:在单词之间跳转
Ctrl+a跳到本行的行首
Ctrl+e则跳到页尾
Ctrl+u删除当前光标前面的文字
ctrl+k删除当前光标后面的文字
Ctrl+y粘贴Ctrl+u或Ctrl+k剪切的内容
Ctrl+L进行清屏操作
Ctrl+w删除光标前面的单词的字符
Alt+d删除光标后面的单词字符 
```

>`Ctrl+左右键`:在单词之间跳转<br/>
>`Ctrl+a`跳到本行的行首<br/>
>`Ctrl+e`则跳到页尾<br/>
>`Ctrl+u`删除当前光标前面的文字<br/>
>`Ctrl+k`删除当前光标后面的文字<br/>
>`Ctrl+y`粘贴Ctrl+u或Ctrl+k剪切的内容<br/>
>`Ctrl+L`进行清屏操作<br/>
>`Ctrl+w`删除光标前面的单词的字符<br/>
>`Alt+d`删除光标后面的单词字符<br/>
